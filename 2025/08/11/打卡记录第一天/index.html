每周打卡总结（8/2 - 8/9）

在前段时间，我有了个决定和计划——在一个地方记录我的每日运动、学习、兴趣爱好的打卡。
经过一些尝试，我放弃了找人、创建公众号等方式。
最终决定：使用 ChatGPT 归纳总结 + 提供模板 + OneNote 记录 的方式。

从 8 月 1 日（周五） 开始，到现在已经过去一周。
途中我发现，工作、生活、学习、娱乐的平衡是我比较焦虑的部分。
ChatGPT 安慰我：我已经出发了，开始行动了，这就很棒了。

整体情况

  • 打卡记录
    • 7 天无空白日。
    • 每天至少完成 2 项打卡任务。
  • 运动
    • 有氧：每天 ≥ 30 分钟。
    • 无氧：仅周三晚上缺席。
  • 吉他
    • 学习 1.5 个月。
    • 仅剩小 F、大 F 未学。
    • G 和弦按法不连贯，中指常悬空（因不按 5 弦 2 品）。
    • 想交弹唱作业，但录视频总不满意，一直拖着。
  • 日语
    • 每天练字帖、抄单词。
    • 买的练习题只写了一点点，动力不足。
  • 冥想
    • 助眠身体扫描冥想尝试失败。
    • 依旧喜欢睡前刷手机。
    • 11 点前睡觉目标未达成。
    • 已将冥想移出打卡计划，轻松不少。

工作与学习

  • 本周工作不忙，有很多学习新知识的机会。
  • 除了摸鱼，我开始了：
    • 网页开发（Node.js 前端 + 后端）
    • React
    • UE C++ 编程
  • 学习进展
    • Node.js 实现简单前后端交互。
    • 重新启动 HEXO 页面并购买域名。
    • 遇到备案问题：
      • 腾讯云要求有云服务器实例才可备案。
      • 云服务器新人价已无。
      • 已有其他平台服务器无法备案。
      • 打算 60 天后将域名转到 贝锐 试试。
    • C++ 方面：
      • 把常用功能写成插件，熟悉语法。
      • 继续写 Pawn 插件 与 Util 插件。
      • 想找一些 UE C++ 开源项目练手。

生活与娱乐

虽然已经不在上周打卡范围里了，还是想写一写昨天

  • 周日与母亲去电影院看了 《浪浪山小妖怪》：
    • 孩子们开心大笑，成年人则像照镜子（哈哈）。
  • 随后逛了 伊藤洋华堂，然后在附近吃了回转寿司。
  • 当天也弹了十几分钟吉他，状态不错，但录视频一紧张就拉胯。
